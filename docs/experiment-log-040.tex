\bb{实验日志：}\hyperref[chap:AUTHOR-kain-pathos-crow]{Kain Pathos Crow教授}，使用\hyperref[chap:SCP-040]{SCP-040}来为“\hyperref[chap:COMP-olympia-project]{奥林匹亚计划}”获取原材料。

\begin{itemize}
\item \bb{日期：}13\slash 12\slash 2008
\end{itemize}

\bb{实验前纪录：}\\
\ii{我认为是时候开始利用040的能力了，或者至少让她能够控制她的能力。她不可能永远依靠\hyperref[chap:SCP-148]{SCP-148}的发饰品。我们的理论认为当她长大后她的力量会成倍数增长，有可能她无意识的心灵感应会失去控制。}

\ii{现在，她似乎并不能完全依靠其自身意识运用她的能力，在她使用能力的时候会让她的潜意识接管她的身体。她说那样很容易做到，但是她并不是经常知道她能力所影响的东西将会变成什么，或者有事甚至不知道她正在做什么。}

\ii{我要让她完成一些练习。我们将从小物件开始，例如鹅卵石，零碎杂物之类的东西，她将要试图将那些东西变成我们想要的东西，例如甲壳动物或者昆虫。}

\bb{实验后纪录：}\\
\ii{嗯……那本可能会更好。}

\ii{实验一开始并不顺利。040在那么多人面前表现紧张，并且那些忧虑反映到了她创造的东西上面。大部分创造物都是凶恶的，时常朝那些冒险站在它们身边的人露出獠牙、尖爪和毒囊，即使040尽了最大努力。大部分实验我们都出于安全考虑而停止了，040对此并不高兴。}

\ii{但是她在实验最后有了些进步，尽管她仍然不能控制整个过程，而且在她展示能力的时候显得情绪反复无常而无条理。不过，这并不是最糟糕的情况。她在监督下的第一次尝试表现不错。离她能做出我要求的东西应该不会很久。她只是需要更多练习。}

\begin{itemize}
\item \bb{17\slash 12\slash 2008}
\end{itemize}

\bb{实验前纪录：}\\
\ii{心理评估进行了，并且其结果不幸被猜中了。显然，她对她的能力患有心理压抑。这唯一原因是因为她需要通过心灵感应来施展她的能力，并在一个可接受的情况来使用能力，但是即使这样做她仍然会对她所做的感到后悔并立刻抑制她自己的记忆。这点会被开发成可控制的反应。关于为什么她会在使用这些能力时感到焦虑也有许多被压抑着的记忆。显然，关于这点我们所拥有的唯一线索是她过去的监护人或者其父母，但是这两者现在都已经过世了。因此，虽然我们能想出如何打破这一阻碍，我们仍将通过增加她被允许使用能力的场合来移除使用她能力时所表现出的耻辱感这种其自身的\bb{限制}。}

\bb{实验后纪录：}\\
\ii{障碍仍然存在，这使得我们的实验和训练变得毫无价值。但是，如果我们在她的心理测试后立刻停止实验和训练，以她的年纪和情感经验的不足，她可能会认为我们停止的原因是因为她犯下了一些错误，这将导致其丧失自尊、孤独以及背上更多的心理包袱，这反过来会导致更多关于她和她的能力的事故，对任何参与的人造成更多的不利影响。因此现在实验继续进行，但我们将准备更多的心理研究并探寻是什么引起了这种心理压抑。}
